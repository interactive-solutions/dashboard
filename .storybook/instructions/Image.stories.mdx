# Image

We are using the image component from Next.js, [next/image](https://nextjs.org/docs/api-reference/next/image).

Demo for [Image](/story/components-general-image--default).

## Domains

When using an external URL, we must add it to domains in `next.config.js`. To control this within different environments we should append this to our space separated `process.env.NEXT_PUBLIC_IMAGES_DOMAINS`.
